HTML Study Notes

HTML Tags

Paragraphs
For adding a paragraph of text we can use the<p></p>tag.
The browser will add a space before and after the content of the paragraph.
Headings
We can use the <h1></h1> tag for the most important heading of the page.
For the next level of heading we have <h2></h2>, <h3></h3>, <h4></h5>, and <h6></h6>.
Quote
Add a small quotation embedded within other content by using <q></q>.
Navigations
Using <nav></nav>to add navigations of the page, this will contain the links to other pages of the website.
Links
For creating a link on our webpage we use the anchor tag <a></a>.
The anchor tag is a paired tag and the content we want to display as a link is added between the opening <a> and the closing </a>.
For providing the location where the anchor tag will link-to we use the href attribute.
